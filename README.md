# 📊 Calculadora de Tiempos — Marketing y Comunicación
### Dimex A.C. · Desarrollos de Ideas Mexicanas

Herramienta web interactiva para planear la carga de trabajo semanal del área de comunicación y marketing, evitando la saturación del equipo.

---

## ✨ Funcionalidades

- **Calculadora de horas** — selecciona tareas por categoría con botones +/− y el total se calcula automáticamente
- **Semáforo de carga** — barra de progreso que indica si la carga es manejable, está al límite o representa riesgo de saturación
- **Desglose por área** — gráfica de barras que muestra en qué categoría se concentra el tiempo
- **Datos del colaborador** — campo para nombre y horas de jornada semanal
- **Selector de periodo** — elige fechas manualmente o usa los atajos: esta semana, próxima semana, quincena o mes
- **Generación de reporte** — resumen imprimible con detalle por tarea, estado de carga y opción de guardar como PDF
- **Tabla de referencia** — tiempos mínimos, típicos y máximos para 25 tipos de tareas

---

## 🖨 Generar un reporte

1. Ingresa el nombre del colaborador y las horas de jornada
2. Selecciona el periodo (o usa los atajos rápidos)
3. Agrega las tareas planeadas con los botones + / −
4. Haz clic en **Generar reporte**
5. Desde el modal, usa **Imprimir / Guardar PDF**
